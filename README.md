# Girls' Frontline Mosin-Nagant — STALKER GAMMA Companion Mod

A fan-made mod that adds **Mosin-Nagant** from *Girls' Frontline* as a standalone NPC companion in *S.T.A.L.K.E.R. GAMMA*.

She spawns at the Rookie Village (Cordon), is recruitable as a companion, and has some custom voice lines, dialog, and combat barks.

---

## Features

- Invulnerable companion NPC with recruit/dismiss dialog
- Japanese voice lines
- Integrated with the GAMMA companion system
- Spawns automatically on a new game or existing save at `esc_smart_terrain_2_12` (Rookie Village)
- Returns to base after dismissal

---

## Requirements

- [S.T.A.L.K.E.R. GAMMA](https://www.moddb.com/mods/stalker-gamma) (latest version)
- Mod Organizer 2 (included with GAMMA) recommended
- You can try with base Anomaly but will need DLTX and DXML for it to work (haven't tested myself)

---

## Installation

1. Download the mod archive as .zip
2. Open Mod Organizer 2
3. Drag the archive into the MO2 window, or use **Install a new mod from archive**
4. Enable the mod in the left panel
5. Place it near the bottom of your load order (after GAMMA base mods)
6. Launch the game — Mosin will spawn automatically

---

## In-Game

She spawns at the **Rookie Village** in the Cordon.
Can be recruited through dialog, and she *should* be invulnerable.

---

## Compatibility

Should be compatible with most GAMMA mods. The mod uses DLTX and DXML, so it does not overwrite any vanilla files. If you encounter a conflict with another NPC or dialog mod, load this mod after it in MO2.
